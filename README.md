# 企業向け一括DM送信システム (Enterprise Bulk DM Sender)

**「メールを送る作業」ではなく、「売上を作る仕事」に集中できる環境を実現します。**
営業・マーケティング部門で発生する「メール配信」「送信履歴管理」「テンプレート切替」などの定型業務を、Google Apps Script（GAS）でワンクリック自動化するシステムです。

## ✨ 特徴
- Google Workspace環境だけで低コストに導入可能
- 業界・顧客セグメントに合わせた文面（テンプレート）を自動切替
- 配信履歴・エラー管理まで自動化し、人的ミスを徹底排除

## 📷 システム連携フロー

```text
[ 顧客リスト ] (Google Sheets)
      ↓
[ 業界判定 ＆ テンプレート選択 ] (Google Docs API)
      ↓
[ メールの自動生成 ＆ 一括送信 ] (Gmail API)
      ↓
[ 送信履歴・ステータスの自動記録 ] (Google Sheets)
      ↓
[ 配信結果のレポート自動通知 ] (Slack / Email)
```

📌 導入ベネフィット (Before / After)
【Before】担当者依存のアナログ運用

1件ずつの手作業による文面作成と送信

宛先間違い、コピペミス、同じ企業への二重送信リスク

送信後の履歴入力や集計に多大な工数がかかる

【After】GASによる業務自動化

✅ ワンクリックで対象顧客へ一括配信

✅ 業界・セグメントごとにドキュメントテンプレートを自動切替

✅ 配信履歴の自動記録と二重送信の徹底ブロック

✅ 管理者への配信結果自動レポート

🚀 実運用に耐えうる技術・セキュリティ設計
大量送信でも停止しにくい安全設計: GASの送信上限や実行時間制限を監視し、クラッシュを未然に防ぐ安全設計。

セキュリティ・誤操作対策: 許可されたアカウント以外からのマクロ実行を弾く権限チェック機能を実装。

例外処理とエラー通知: try-catchによる確実なエラー処理と、Slack/Emailへの通知機能を完備。

保守性の高いオブジェクト管理: テンプレートの追加・変更が設定値の変更のみで完結する拡張性の高いアーキテクチャ。
